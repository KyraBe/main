Proyecto SSDLC

Este proyecto fue creado enteramente en Python usando Microsoft Visual Studio, razón por la cual el archivo principal se llama PythonApplication1.py. Cada archivo auxiliar es importante para la ejecución y funcionamiento de la aplicación.  Además, he dejado Datos.xlsx y Audit.txt para que tengas un ejemplo de cómo deben ser estos documentos para el correcto funcionamiento del programa, así como una matriz de auditoría.

Para el inicio de sesión, existen actualmente tres cuentas de usuario con los siguientes nombres de usuario y contraseñas: Admin - Administrador; Asistente - Asistente; Vendor - Vendedor.

Tenga en cuenta que las respuestas distinguen entre mayúsculas y minúsculas. Siempre que introduzca una opción, por favor, asegúrese de respetarla, de lo contrario puede que no funcione correctamente. 

Si tienes alguna pregunta, sobre cómo traducir algo por ejemplo, no lo dudes.
